# Tarea: Tipos de Datos e Identificadores en Python

Este repositorio contiene un programa sencillo en Python desarrollado para practicar:
* Uso de identificadores descriptivos (snake_case).
* Implementación de tipos de datos (string, integer, float, boolean).
* Uso de comentarios y documentación.

## Cómo ejecutarlo
Solo necesitas tener instalado Python y ejecutar el archivo `gestion_datos.py`.
